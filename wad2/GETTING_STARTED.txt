================================================================================
                          TROUBLESHOOTING
================================================================================

Q: Port 4200 already in use?
A: ng serve --port 4300

Q: Dependencies not installed?
A: npm install

Q: Clear application data?
A: Open DevTools → Application → LocalStorage → Delete keys

Q: Lost password?
A: Clear localStorage and re-register

Q: Can't edit profile?
A: Make sure you're logged in and localStorage is enabled

Q: Page shows "No user data found"?
A: Clear localStorage and register a new account
